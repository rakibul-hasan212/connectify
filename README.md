
# Connectify – Real-Time Flutter Chat Application

Connectify is a modern real-time chatting application built with Flutter and Firebase. The app is designed with a scalable and production-ready architecture to provide a smooth messaging experience similar to popular messaging platforms like WhatsApp and Messenger.

The application uses Flutter for cross-platform mobile development, allowing a single codebase for Android and future iOS support. Firebase Authentication is used for secure user login and registration, while Cloud Firestore is used as the real-time database for storing users, chat rooms, and messages.

The app supports real-time messaging using Firestore snapshot listeners, which instantly update conversations without refreshing the screen. Users can search other users, start new conversations, and continue previous chats from the recent chat list.

Connectify follows clean and organized architecture with GetX State Management for reactive UI updates, navigation, dependency management, and controller-based business logic separation. The project also uses reusable widgets, utility classes, and modular folder structure to maintain clean and maintainable code.

Important features implemented in the application include:

* Real-time one-to-one messaging
* Smart chat room generation
* Recent chat list system
* User search functionality
* Auto scroll to latest message
* Smart timestamp formatting
* Online/offline user status
* Reusable custom widgets
* Reactive UI with GetX
* Firebase Firestore integration
* Secure authentication system

The application is developed with performance, scalability, and real-world app architecture in mind. Firestore subcollections are used for efficient message storage, while chat metadata is maintained separately for fast recent chat loading.

Connectify is a proper Flutter project that demonstrates practical knowledge of state management, Firebase backend integration, real-time data handling, scalable database design, and modern mobile app development techniques.
